# solowsimulation
Solow-Swan Growth Model's simulation

## Usage

Install dependencies and open this repository in a Jupyter environment:

```bash
pip install -r requirements.txt
```

Then run:

```python
from solow_simulation import simulazione_solow

# Use the interactive widget in a notebook
# or run the script directly in Jupyter:
# %run solow_simulation.py
```

This script provides an interactive slider interface for the Solow model parameters:
- `s`: savings rate
- `n`: population growth rate
- `g`: technological progress rate
- `delta`: depreciation rate
- `alpha`: capital share in production
